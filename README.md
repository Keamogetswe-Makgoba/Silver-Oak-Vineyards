# 🍇 Silver Oak Vineyards Website

A modern, responsive vineyard landing page built with HTML and CSS.  
This project showcases a luxury wine brand UI with a hero section, product cards, and a clean footer.

## ✨ Features

- Responsive hero section with full-screen background image  
- Email signup form UI  
- Wine product cards using Flexbox  
- Modern footer design  
- Clean vineyard-inspired aesthetic  
- Mobile-friendly layout
